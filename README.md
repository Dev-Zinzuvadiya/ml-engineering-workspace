# ML Engineering Workspace

This repository contains my machine learning learning materials, practice projects, experiments, notes, datasets, and other related work.

I use it to keep everything in one place while learning and trying out different ideas.

## Repository Structure

| Directory              | Purpose                                    |
| ---------------------- | ------------------------------------------ |
| analytics              | Data analysis and exploration              |
| configs                | Configuration files and setup resources    |
| datasets               | Datasets used for practice and experiments |
| django_projects        | Web-related projects and experiments       |
| ml_docs                | Notes, references, and learning materials  |
| statistics             | Statistics exercises and concepts          |
| all_around_experiments | General experiments and testing            |

## Notes

Most folders contain learning projects, experiments, or reference materials, so the code and structure may vary from one project to another.

This repository is mainly a place to store and track my work as I learn and build things.
